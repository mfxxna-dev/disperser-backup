# Disperser Studio 🎙️

Professional Roblox Audio Asset Management Suite.

## Project Structure

- `/frontend`: Vite + React + TypeScript (Studio UI)
- `/backend`: Node.js + Express (Roblox Proxy & YT Downloader)

## Setup & Development

### 1. Backend
```bash
cd backend
npm install
npm run dev
```

### 2. Frontend
```bash
cd frontend
npm install
npm run dev
```

## Production Build

### Frontend
Run `npm run build` in the `/frontend` directory. This generates a `dist` folder.

### Backend
The backend is a standard Node.js server. For production, you can run `npm run build` to compile TypeScript, then `npm start` to run the compiled JS.
